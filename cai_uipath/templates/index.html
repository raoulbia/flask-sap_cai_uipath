SAP CAI Chatbot with UiPath Integration

PoC UiPath Integration with Flask-Powere SAP Conversational AI Chatbot.
This chatbot allows users to search for products on Amazon. The chatbot will return a list of
the top 10 results retrieved from Amazon. This chatbot has been implemented using the SAP Conversational AI platform and the Python Flask web framework.
The Flask application is hosted on github.
